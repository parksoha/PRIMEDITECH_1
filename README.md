# PRIMEDITECH WordPress Theme

## 테마 정보

- **테마 이름**: PRIMEDITECH Custom
- **버전**: 1.0
- **작성자**: PRIMEDITECH
- **설명**: PRIMEDITECH 홈페이지 리뉴얼 테마 - 다국어 지원 (한국어, 영어, 중국어, 일본어, 독일어, 프랑스어)
- **WordPress 버전**: 5.0 이상
- **PHP 버전**: 7.4 이상

## 주요 기능

### 🌐 다국어 지원
- 한국어 (기본)
- 영어
- 중국어
- 일본어
- 독일어
- 프랑스어

### 📱 반응형 디자인
- 모바일 우선 설계
- 태블릿 및 데스크톱 최적화
- 터치 친화적 인터페이스

### 🎨 현대적인 디자인
- 깔끔하고 전문적인 UI/UX
- 의료기기 업체에 적합한 디자인
- 접근성 고려 (WCAG 2.1 준수)

### ⚡ 성능 최적화
- 빠른 로딩 속도
- 이미지 최적화
- CSS/JS 압축

### 🔧 WordPress 기능
- 커스텀 포스트 타입 지원
- 위젯 영역
- 메뉴 지원
- 커스터마이저 지원

## 설치 방법

1. WordPress 관리자 패널에 로그인
2. 외모 > 테마로 이동
3. "테마 업로드" 클릭
4. 테마 ZIP 파일 업로드
5. "활성화" 클릭

## 권장 플러그인

### 필수 플러그인
- **GTranslate**: 다국어 번역 기능
- **Channel.io**: 실시간 채팅 위젯
- **WP Statistics**: 방문자 통계
- **Contact Form 7**: 문의 폼

### 선택 플러그인
- **Greenshift Animation**: 고급 애니메이션 효과
- **Yoast SEO**: SEO 최적화
- **Elementor**: 페이지 빌더
- **WP Rocket**: 캐싱 및 성능 최적화

## 테마 설정

### 메뉴 설정
1. 외모 > 메뉴로 이동
2. 새 메뉴 생성
3. "Primary Menu" 위치에 할당
4. "Footer Menu" 위치에 할당

### 위젯 설정
1. 외모 > 위젯으로 이동
2. 사이드바 및 푸터 위젯 영역에 위젯 추가

### 커스터마이저
1. 외모 > 사용자 정의로 이동
2. 색상, 로고, 메뉴 등 설정

## 파일 구조

```
prmtec-theme/
├── style.css              # 메인 스타일시트

├── header.php             # 헤더 템플릿
├── footer.php             # 푸터 템플릿
├── functions.php          # 테마 함수
├── single.php             # 단일 포스트 템플릿
├── page.php               # 페이지 템플릿
├── archive.php            # 아카이브 템플릿
├── search.php             # 검색 결과 템플릿
├── 404.php                # 404 에러 템플릿
├── comments.php           # 댓글 템플릿
├── sidebar.php            # 사이드바 템플릿
├── searchform.php         # 검색 폼 템플릿
├── screenshot.png         # 테마 스크린샷
├── css/                   # CSS 파일들
│   ├── common.css
│   ├── main.css
│   ├── style.css
│   ├── responsive.css
│   └── ...
├── js/                    # JavaScript 파일들
│   ├── main.js
│   └── gtranslate.js
├── img/                   # 이미지 파일들
│   ├── mainlogo.png
│   └── ...
└── sub/                   # 서브 페이지 템플릿
    ├── about.php
    ├── services.php
    ├── products.php
    └── ...
```

## 커스터마이징

### 색상 변경
1. 외모 > 사용자 정의 > 색상
2. 기본 색상: #009245 (녹색)

### 로고 변경
1. 외모 > 사용자 정의 > 사이트 ID
2. 로고 업로드 (권장 크기: 200x60px)

### 메뉴 설정
1. 외모 > 메뉴
2. 새 메뉴 생성 후 위치 할당
